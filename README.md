# osaka

Package to encode text to osaka dialect.
This is porting of http://www.geocities.co.jp/Hollywood/4078/osaka/osakajs.html

## Usage

```
fmt.Println(osaka.Encode("儲かりますか？"))
```

## Installation

```
$ go get github.com/mattn/osaka/cmd/osaka
```
